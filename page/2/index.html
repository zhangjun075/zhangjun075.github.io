goruntine介绍

发布时间 : 2019-04-18
发布 : 2019-04-18 分类 : go 浏览 :

go runtime、协程底层实现原理、与线程的比较等。系统不知道协程的存在，是完全由用户自己的程序进行调度的，因为是由用户程序自己控制，那么就很难像抢占式调度那样做到强制的 CPU 控制权切换到其他进程线程，通常只能进行协作式调度，需要协程自己主动把控制权转让出去之后，其他协程才能被执行到。
